# Scholar Spend - Personal Finance Tracker 💰

**Description:** A desktop personal finance application featuring a custom Graphical User Interface (GUI) designed to help students track expenses, manage budgets, and analyse transaction history.

## 🛠️ Tech Stack
* **Language:** Java
* **Frontend:** HTML, Java GUI components
* **Environment:** Eclipse IDE

## ✨ Key Features
* **Expense Tracking:** Secure structuring of user financial data using Object-Oriented Programming (OOP) principles.
* **Custom GUI:** Seamless frontend-to-backend integration for smooth user interaction.
* **Budget Management:** Automated budget calculations and history tracking.
